The run-time live football scores system
========================================

Technologies

Development environment:

    Visual Studio 2013 Preview
    SQL server 2012 management tool

Database:

    SQL server 2012
    Redis (using pub/sub pattern for broadcasting a message)

Back-end:

    Entity framework 6.0
    Sharp lite

Front-end:

    The ASP.NET MVC 5
    The ASP.NET Web API
    SignalR
    Angularjs
